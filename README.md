# 🐶 Pet Shop Sales Dashboard

## 📌 Sobre o projeto

Este projeto consiste no desenvolvimento de um dashboard em **Power BI** para análise do faturamento e desempenho de um Pet Shop, simulando um cenário real de Business Intelligence.

O objetivo foi transformar dados brutos em informações relevantes para apoiar a tomada de decisão, aplicando conceitos de ETL, modelagem de dados, visualização e criação de indicadores de desempenho.

---

## 🎯 Objetivos

- Realizar o processo de ETL utilizando Power Query;
- Tratar e padronizar os dados provenientes de diferentes fontes;
- Construir um modelo de dados para análise;
- Criar indicadores (KPIs) para acompanhamento do negócio;
- Desenvolver um dashboard interativo com foco em usabilidade.

---

## 🛠️ Tecnologias utilizadas

- Power BI
- Power Query
- DAX
- Microsoft Excel
- Google Sheets

---

## 📊 Indicadores (KPIs)

- 💰 Faturamento Total
- 🛒 Quantidade de Vendas
- 🐾 Média de Pets Atendidos
- 📈 Evolução do Faturamento
- 📍 Análise por Região

---

## 🔄 Processo de desenvolvimento

Durante o projeto foram realizadas as seguintes etapas:

- Extração de dados do Excel e Google Sheets;
- Tratamento e limpeza dos dados no Power Query;
- Remoção de inconsistências e padronização das informações;
- Modelagem dos dados;
- Criação de medidas em DAX;
- Desenvolvimento das visualizações;
- Construção de um dashboard responsivo para Web e Mobile.
